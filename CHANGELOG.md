# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-02

### Added
- 初始版本发布
- WebSocket 实时通信支持
- Protocol Buffer 协议实现
- 用户认证和会话管理
- 单聊消息收发
- 群聊消息收发
- 离线消息存储和同步
- 消息已读回执
- 消息撤回功能
- 输入状态通知
- 在线状态管理
- PostgreSQL 数据库支持
- MySQL 数据库支持
- Redis 缓存支持
- Docker 容器化部署
- Docker Compose 编排配置
